# React Fox Game

A fun and interactive fox-themed game built with React.

## Features

- Engaging gameplay mechanics
- Responsive design
- Real-time interactions

## Installation

```bash
npm install
```

## Usage

```bash
npm run dev
```

Runs the game in development mode. Open [http://localhost:3000](http://localhost:3000) to play.

## Build

```bash
npm run build
```

Builds the app for production to the `build` folder.

## Test

```bash
npm run test
```

Runs the tests for the game.

## Notes

- Preloading images to avoid flickering
- Timer-based game loop
- LocalStorage scoreboard persistence

## Technologies

- React
- JavaScript
- CSS
- Jest (for testing)
- React Testing Library (for testing)
